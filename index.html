Patron
GitHub repo

Patron is a social media app that allows users to utilize the extensive Metropolitan Museum of Art's API in order to create their own personal art gallery. By utilizing this API a user, or Patron, has full access to the Metropolitan Museum of Art's entire catalogue and they can engage in discussion with other Patrons.

Kindly
GitHub Repo

Kindly is a social media website that allows users to create an account as well as create and post local volunteer opportunities. Once a user has created an account or logged in to their account, they may sign up for volunteer opportunities; post new volunteer events; and comment on each volunteer event that is posted. Users are able to engage in productive discourse and share memories of their experiences as well.
